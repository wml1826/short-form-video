# MediaPlay 项目长期笔记

## 项目概况
- C++ / Qt 媒体播放器，基于 FFmpeg 4.2.2 + SDL2 2.0.10 做音视频解码与渲染。
- 用户用本项目作为技术面试准备材料（2026-07-13 起以“面试官/候选人”角色对练）。

## 音视频同步要点（面试常问）
- fps（视频帧率，本项目 25fps → 每帧 40ms）：无音频流时由 SDL_AddTimer 按 1/fps 驱动显示（videoplayer.cpp:376-388）。
- Hz（音频采样率，44.1kHz）：AAC 每帧 1024 样本 → 1024/44100 ≈ 23.2ms，声卡按此节奏消耗缓冲区。
- 二者都归一化到时间轴，靠 PTS（秒）做同步；有音频时以 audio_clock 为主时钟。
- GOP=12 @25fps ≈ 每 0.48s 一个 I 帧。

# MediaPlay · 短视频播放与社交平台

> 一个基于 **Qt + FFmpeg + SDL2 + OpenCV** 的桌面端短视频应用，配套自研 C++ 后端服务。
> 支持本地播放、在线点播、直播推流/拉流、录制、上传、画中画等完整能力，
> 可作为音视频、网络编程、多线程与客户端架构的综合学习 / 面试项目。

---

##  功能特性

| 模块          | 说明                                                         |
| ------------- | ------------------------------------------------------------ |
|  本地播放    | 基于 FFmpeg 解封装/解码，SDL2 渲染音频，OpenGL 渲染视频帧；支持播放 / 暂停 / 停止 / 进度跳转 |
|  在线点播    | 登录注册后浏览视频列表（按兴趣标签分类）、点击即播、查看个人上传历史 |
|  直播        | 一键开播（RTMP 推流）、实时预览；观众端可拉流观看他人直播间  |
|  录制        | 采集摄像头 / 桌面画面，编码并保存为本地视频文件              |
|  上传        | 选择视频并自动截取封面图，按兴趣标签上传，带上传进度回调     |
|  画中画      | 同时采集摄像头与桌面画面，完成 RGB→YUV420P 转换，叠加显示    |
|  OpenGL 渲染 | 独立的 `MyOpenGLWidget`，用于高效视频帧绘制                  |

##  技术栈与依赖

| 组件         | 版本 / 说明                                               | 位置            |
| ------------ | --------------------------------------------------------- | --------------- |
| Qt           | 5.x（`widgets` / `multimedia` / `network` / `opengl`）    | 需自行安装      |
| FFmpeg       | 4.2.2（随仓库附带）                                       | `ffmpeg-4.2.2/` |
| SDL2         | 2.0.10（随仓库附带）                                      | `SDL2-2.0.10/`  |
| OpenCV       | 用于摄像头/桌面采集与格式转换（**未随仓库**，需自行下载） | 外部依赖        |
| MySQL Client | 服务端持久化依赖                                          | 外部依赖        |
| 运行时 DLL   | 执行所需动态库                                            | `dll/`          |

>  当前工程面向 **Windows** 平台构建（依赖 `winsock`、`opengl32` 等）。

##  使用流程

1. 启动客户端 → 主界面 `PlayerDialog` 可本地选片播放。
2. 点击「在线」→ 登录 / 注册账号。
3. 浏览点播列表，点击视频即可在线播放；也可上传自己的短视频（自动截封面 + 选标签）。
4. 点击「直播」→ 开播后其他人可在直播间列表看到并观看；或作为观众拉流观看。
5. 「录制」与「画中画」用于本地采集摄像头 / 桌面画面并保存或叠加。
